# HELP.Desk
App to manage maintenance calls. This application was created for the purpose of studies.


## Stack

- React Native
- Typescript
- Expo Bare Workflow
- Firebase
- Firebase Auth
- Password recovery with e-mail
- Firebase Firestore with Real Time updates
- And more...


## Features

- User authentication 
- Create new user
- Create new order
- List orders
- Show real time status
- And more...
